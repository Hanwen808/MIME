# Network Measurement Applications

This is a Python implementation for MIME in multi-spread measurement, superhost detection, and super k-persistent spread measurement.

## Project Structure

```
MIME_Cpp/
├── MIME_main.cpp
├── CSE_main.cpp
├── Supersketch_main.cpp
├── rSkt_main.cpp
├── vHLL_main.cpp
├── ipv4.cpp
├── Headers/

│   ├── CSE.h
│   ├── MIME.h
│   ├── Supersketch.h
│   ├── bitmap.h
│   ├── rSkt.h

│   ├── Sketch.h

│   ├── MurmurHash3.cpp
│   ├── Supersketch.cpp
│   ├── bitmap.cpp
│   ├── rSkt.cpp
│   └── vHLL.cpp
```

## Requirements

- Python 3
- numpy >= 1.21.6; matplotlib >= 3.2.1
