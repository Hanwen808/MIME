FPGA implementation of MIME
============

Core File Description
--------------------
*  `bitmap.cpp`: the implementation of a bit array.
*  `murmurhash.cpp`: the implementation of MurmurHash3.
*  `supercube_fpga.cpp`: the implementation of MIME. When compiling MIME, please set the top-function to `supercube_fpga.cpp`.

Compile Tools
--------------------
* compile: Vivado HLS
